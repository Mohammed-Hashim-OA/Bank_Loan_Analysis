🏦 Bank Loan Report Analysis

A comprehensive Bank Loan Analytics project built using SQL and Power BI to monitor loan performance, evaluate lending quality, and generate actionable business insights through interactive dashboards.

This project demonstrates an end-to-end Business Intelligence workflow, transforming raw banking data into meaningful KPIs, trends, and risk indicators that support better lending decisions.

------------------------------------------------------------------------

🚀 Project Overview

Financial institutions process thousands of loan applications every year. Monitoring loan performance, repayment behavior, customer risk, and portfolio health is essential for making informed lending decisions.

This project analyzes a bank's loan portfolio using SQL for business analysis and Power BI for interactive reporting.

------------------------------------------------------------------------

🎯 Objectives

-   Analyze overall loan portfolio performance
-   Track funded and received loan amounts
-   Measure lending efficiency using KPIs
-   Compare Good Loans vs Bad Loans
-   Analyze loans by geography, purpose, employment length, ownership,
    and term
-   Build an executive dashboard for business users

📚 Banking Terminologies

-   MTD (Month-to-Date): Performance from the beginning of the current month up to the current date.
-   MoM (Month-over-Month): Compares current month performance with the previous month.
-   DTI (Debt-to-Income Ratio): Measures a borrower's debt relative to income.
-   Good Loan: Loan status is Current or Fully Paid.
-   Bad Loan: Loan status is Charged Off.
